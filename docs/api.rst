API reference
=============

All functions accept NumPy arrays (converted to contiguous float64 on
entry) and pandas objects (Series in → Series out with the index
preserved; DataFrame in → DataFrame out for the 2-D functions).  The R
package ``fastroll`` mirrors this API; see ``?rolling_mean`` etc. in R.
